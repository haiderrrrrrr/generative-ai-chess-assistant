# Generative AI Chess Agent

An intelligent, Python-based chess bot that utilizes Generative AI concepts and heuristic search algorithms to evaluate board states and determine optimal counter-moves against a human player.

## 🚀 Key Features

* **Generative Board Evaluation**: Employs AI search techniques to generate decision trees and calculate the highest probability of success based on piece value and positioning.
* **Interactive Gameplay**: Allows a user to play a full game of chess directly against the AI.
* **Heuristic Analysis**: Tracks complex rules like castling, en passant, and checkmate conditions algorithmically.
* **Automated Move Generation**: Python engine processes millions of potential move permutations to optimize the bot's strategy.

## 🛠️ Technology Stack

* **Language**: Python 3.x
* **Domain**: Artificial Intelligence / Game Theory
* **Dependencies**: Standard Python Libraries (See \chess.py\ imports)
